Completed Projects

Little Rat

Little Rat is a Java application I made in 2017. It maps each of 100,000 English words to the word which sounds most similar to it. "Most similar" is defined by a scoring system and the same global alignment algorithm used to compare sequences of DNA. Little Rat includes a GUI to test its word mappings. It can be used for both lyric composition and speech recognition.
